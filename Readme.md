* Project：Cloud-X
* OfficialWebsite：http://mongo.ucaner.cn

### 简述:
某宝入手一个树莓派2B，打算搭建KODI,或者其他好玩的东西，特次搭建一个Web项目
来通过PI4J来实现远程控制树莓派 下达操作指令
后期申请机智云搭建内网穿透等实现远程控制
提供开放API接口(接口鉴权) 基于Dubbox [restful webservice]
